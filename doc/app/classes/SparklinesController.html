show()

Use this type of method for sparklines that can be cached. (Doesn‘t work with the helper.)

To make caching easier, add a line like this to config/routes.rb: map.sparklines "sparklines/:action/:id/image.png", :controller => "sparklines"

Then reference it with the named route: 

  image_tag sparklines_url(:action => 'show', :id => 42)
