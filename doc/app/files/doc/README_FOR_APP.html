GLSAMaker v2

GLSAMaker is a core tool for the Gentoo Security Team. It is used to track, create, edit and review Security announcements and drafts.

Project website: security.gentoo.org

Needed libraries

  • Rails 2.3.2
  • Nokogiri
  • FasterCSV
  • Sparklines
